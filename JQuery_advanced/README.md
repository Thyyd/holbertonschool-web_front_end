# Projet Javascript Advanced

Dans ce projet, nous allons apprendre différentes utilisations de JQuerry, comme :
- Comment charger une JQuerry d'un CDN sur une page
- Comment créer des éléments de DOM avec JQuerry
- Comment ajouter, modifier, ou encore mettre des événements sur des éléments
- Comment créer de la pagination
- Comment envoyer des requêtes POST, GET, etc...
