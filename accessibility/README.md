# holbertonschool-headphones

Dans ce projet, nous allons apprendre quels sont les niveaux d'accessibilité d'un site et comment rendre un site plus accessible.

## Objectifs

- Apprendre quelles propriétés ajouter pour :
  - Permettre une navigation simplifiée en utilisant le clavier.
  - Créer des skip links
- Comprendre comment utiliser des ARIA

## Technologies

- Langage HTML
- Langage CSS
