# Projet SASS-SCSS

Ce projet va nous permettre d'apprendre progressivement l'utilisation du SASS et du SCSS

## Objectifs

- Apprendre à utiliser les Developers Tools sur Chrome pour :
    - Apprendre comment détecter les 'Error 404'
    - Apprendre comment bloquer des requêtes
    - Apprendre comment modifier le HTML et le CSS à l'aide de l'onglet 'Elements'
    - Savoir quelle quantité de JavaScript ou de CSS est utilisée sur une page
    - Apprendre comment déplacer des éléments sur une page web
