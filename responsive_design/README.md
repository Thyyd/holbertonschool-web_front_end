# Projet responsive design

Ce projet va nous permettre d'apprendre comment rendre un site responsif, faire en sorte que la page web ait une bonne résolution et une bonne lisibilité peu importe la taille de l'écran ou la résolution de l'écran, tout en assurant une bonne ergonomie.

## Objectifs

- Apprendre à rendre un site responsif pour mobile dans un premier temps
- Voir comment adapter le côté responsif pour pouvoir le faire également sur des tablettes

## Technologies

- Code HTML Fourni
- Code CSS Fourni, puis modifié
